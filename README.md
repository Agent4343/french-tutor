An interactive French learning app with real-time speech recognition and pronunciation feedback.

## Features

- **Phrase Practice**: Learn essential French phrases organized by difficulty level
- **Speech Recognition**: Speak French and get your pronunciation analyzed
- **Pronunciation Scoring**: Receive instant feedback on how well you pronounced phrases
- **Native Audio**: Listen to correct pronunciation using text-to-speech
- **Pronunciation Tips**: Learn about French pronunciation rules like nasal vowels, silent letters, and liaisons
- **Conversation Practice**: Practice real-world scenarios like ordering at a cafÃ© or asking for directions

## How It Works

1. **Listen**: Click the speaker button to hear the correct French pronunciation
2. **Speak**: Click the microphone and say the phrase in French
3. **Learn**: Get instant feedback with a score and helpful pronunciation tips

## Pronunciation Rules Covered

- Silent endings (d, t, s, x, z, p)
- Nasal vowels (an/en, on, in/ain)
- The French R sound
- The French U vs OU sounds
- Liaison between words
- EU and OI sounds

## Tech Stack

- Next.js 14
- React 18
- Web Speech API (Speech Recognition & Synthesis)
- CSS-in-JS styling
